iTrackers - User Roadmap

Q4 2024: Initial Launch

Platform Launch

  • Connect your TON wallet to explore the platform.
  • Search for data breaches by email, name, or hash.
  • Verify if your passwords have been compromised.

Q1 2025: New Features & Enhancements

Enhanced Search Capabilities

  • Expanded database for even more comprehensive data breach searches.
  • New search filters to refine your queries.
  • Improved user interface for easier navigation.

Password Strength Checker

  • Check the strength of your passwords before using them.
  • Suggestions for creating stronger passwords.

Q2 2025: Community and Security

Community Features

  • User forums and discussion boards.
  • Peer-to-peer support and shared experiences.

Enhanced Security

  • Two-factor authentication (2FA) for added security.
  • Continuous security updates to protect user data.

Q3 2025: Advanced Tools

API Access

  • API access for developers and businesses to integrate data breach searches.

Custom Alerts

  • Set up custom alerts for specific data breaches.
  • Receive real-time notifications if your data is compromised.
